# ShineCore Launcher

Лаунчер для проекта **ShineCore** — клиент для подключения и удобного запуска
моего Minecraft‑сервера. Сейчас это прототип интерфейса без игровой логики.

## Технологии
- **Frontend:** Vue 3 + Vite + TypeScript + Pinia
- **Backend:** Go (Wails)

## Назначение
Проект создаётся для моего Minecraft‑сервера **ShineCore** и будет развиваться
по мере готовности серверной части.
