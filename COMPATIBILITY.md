# Совместимость ShineCore Launcher с Windows

## Поддерживаемые версии Windows

### Основные требования
- **Windows 10** (все версии, включая Home, Pro, Enterprise)
- **Windows 11** (все версии)

### Архитектура процессора
- **AMD64 (x64)** — основная поддержка
- **ARM64** — поддерживается (экспериментально)

## Системные требования

### Минимальные требования
- **ОС**: Windows 10 (версия 1607 или новее)
- **WebView2 Runtime**: Обязательно установлен
  - Обычно входит в Windows 11 по умолчанию
  - Для Windows 10 требуется установка (автоматически предлагается при первом запуске)
  - Скачать можно с: https://go.microsoft.com/fwlink/p/?LinkId=2124703
- **RAM**: 2 GB (рекомендуется 4 GB для комфортной работы)
- **Свободное место на диске**: 
  - ~100 MB для самого лаунчера
  - Дополнительно ~2-5 GB для установки Minecraft и модов
  - Java runtimes загружаются автоматически (~200-300 MB каждая версия)

### Рекомендуемые требования
- **ОС**: Windows 11 или Windows 10 версии 1903 (19H1) или новее
- **RAM**: 8 GB или больше
- **Процессор**: 2+ ядра, частота 2.0+ GHz
- **Интернет**: Стабильное соединение для загрузки файлов

## Технические детали совместимости

### Используемые технологии
1. **Wails v2.11.0**
   - Требует Windows 10+ (это ограничение самого фреймворка)
   - Использует WebView2 для рендеринга интерфейса

2. **Windows API функции** (используются напрямую)
   - `GlobalMemoryStatusEx` (kernel32.dll) — определение объема RAM
     - Совместимо с: Windows XP+
     - В вашем случае ограничивается требованиями Wails (Windows 10+)
   
   - `CREATE_NO_WINDOW` флаг (syscall.SysProcAttr) — скрытие консоли Java
     - Совместимо с: Windows XP+
     - Используется для запуска `javaw.exe` без всплывающих окон

3. **Java Runtime Environment (JRE/JDK)**
   - Java 8, 17, 21 — загружаются автоматически в папку игры
   - Не требуется системная установка Java
   - Работает с Windows 10+ (поддерживается Oracle/OpenJDK)

### Что НЕ поддерживается

❌ **Windows 7/8/8.1**
- Wails v2 не поддерживает эти версии
- WebView2 требует Windows 10+

❌ **Windows XP/Vista**
- Устаревшие версии, не поддерживаются современными технологиями

❌ **Windows Server без Desktop Experience**
- Требуется GUI для работы лаунчера

❌ **Windows 10 версии ниже 1607**
- Минимальная поддерживаемая версия Wails v2

## Проверка совместимости

### Как проверить версию Windows:
1. Нажмите `Win + R`
2. Введите `winver` и нажмите Enter
3. Проверьте версию (должна быть Windows 10 версии 1607+ или Windows 11)

### Как проверить наличие WebView2:
1. Откройте командную строку (cmd)
2. Выполните:
   ```
   reg query "HKEY_LOCAL_MACHINE\SOFTWARE\WOW6432Node\Microsoft\EdgeUpdate\Clients\{F3017226-FE2A-4295-8BDF-00C3A9A7E4C5}"
   ```
3. Если ключ существует — WebView2 установлен

Или просто запустите лаунчер — если WebView2 отсутствует, будет показано предложение установить его.

## Потенциальные проблемы и решения

### Проблема: "WebView2 not found"
**Решение**: Установите WebView2 Runtime с официального сайта Microsoft или дождитесь автоматической установки.

### Проблема: "Access denied" при установке Java
**Решение**: Запустите лаунчер от имени администратора или проверьте права доступа к папке установки игры.

### Проблема: Лаунчер не запускается на старых версиях Windows 10
**Решение**: Обновите Windows до версии 1903 или новее через Центр обновления Windows.

### Проблема: Низкая производительность на старых компьютерах
**Решение**: 
- Закройте другие приложения
- Убедитесь, что достаточно свободной RAM
- Проверьте наличие обновлений Windows

## Установка и запуск

### Первая установка
1. Скачайте установщик лаунчера
2. Запустите установщик (требуются права администратора)
3. При первом запуске может быть запрошена установка WebView2 — согласитесь
4. Лаунчер готов к работе

### Первый запуск игры
1. Лаунчер автоматически определит требуемую версию Java по версии Minecraft
2. Если Java не найдена локально — начнётся автоматическая загрузка из интернета
3. После установки Java и файлов игры можно запустить игру

### Офлайн работа
- Лаунчер может работать офлайн после первого запуска
- Используется кешированный манифест с сервера
- Если сервер недоступен — игра запустится с настройками из локального конфига

## Автоматические проверки

Лаунчер автоматически:
- ✅ Проверяет наличие нужной версии Java для версии Minecraft
- ✅ Загружает Java, если она отсутствует в папке игры
- ✅ Кеширует манифест для офлайн работы
- ✅ Очищает старые файлы при смене версии игры/загрузчика
- ✅ Использует `javaw.exe` вместо `java.exe` для скрытия консоли
- ✅ Применяет флаги скрытия окна для всех Java процессов

## Поддержка

Если у вас возникли проблемы с запуском на поддерживаемой версии Windows:
1. Проверьте, что установлена актуальная версия лаунчера
2. Убедитесь, что WebView2 Runtime установлен
3. Проверьте версию Windows (должна быть 10 1607+ или 11)
4. Попробуйте запустить от имени администратора
